IBRAQ Store — النسخة المحسّنة

الملفات الرئيسية:
- index.html: واجهة المتجر
- admin.html: لوحة الإدارة
- style.css / app.js: تصميم وتشغيل المتجر
- admin.css / admin.js: تصميم وتشغيل لوحة الإدارة

بيانات دخول لوحة الإدارة الافتراضية:
admin
admin123

ما تم تحسينه:
- إصلاح اختفاء شعار إبراق من رأس الصفحة.
- إصلاح إضافة المنتجات للسلة وإغلاق نافذة الكمية.
- منع تجاوز كمية المخزون.
- تعطيل زر المنتج غير المتوفر.
- منع الزوم وتأثيرات التكبير والوميض على الهاتف.
- تحسين سرعة البحث وتحميل الصور.
- ضغط الصور المرفوعة من لوحة الإدارة تلقائياً لتقليل اللاك.
- إظهار رسالة واضحة عند امتلاء مساحة حفظ المتصفح.
- تحسين التصميم العام للمتجر ولوحة الإدارة.
- تحسين عرض الهاتف والمسافات والأزرار والسلة.
- إضافة حماية للنصوص المعروضة داخل بطاقات المنتجات.

مهم:
هذه النسخة ما زالت تحفظ المنتجات والإعدادات داخل متصفح الجهاز (LocalStorage).
للحفظ الدائم على كل الأجهزة يجب ربطها لاحقاً بقاعدة بيانات Supabase.
قبل الربط، استخدم زر تصدير البيانات في لوحة الإدارة للاحتفاظ بنسخة احتياطية.
